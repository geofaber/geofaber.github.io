Geo Faber's collection of example programs and test data

This collection includes a number of C language programs and functions. Most of those use or are related to the open‑source Nemo Library of Round-world Computational Primitives.

Neither the Library nor the demo programs have any external "dependencies". The code is standard‑conforming C and is expected to be fully portable to any contemporary 64-bit C compiler. They require no "build-yools" nor do they use of any "integrated development environment" or "project" setup. Their source is not resident on any "collaborative programming" platform. A set of simple operating system specific build scripts for library and for the demo programs is provided. These can be used as models for other operating system and compiler specific compile/link scripts.

The programming style of Library source code - and especially that of the code included in this collection - assumes the user is not primarily a computer programmer, but instead a professional of some application‑specific technical discipline, possessing an adequate level of knowledge of C language and general principles of computer programming.

UniSpherical Coordinate System: example programs and data

The principal data set used by the programs in this section consists of close to two million points, distributed over the planetary surface with local density matching the system activity level of a large class of application systems that deal with the "where of things" related the planetary human population.

The data set is distributed by the Department of Combinatorics and Optimization at the University of Waterloo as a text file of 1904711 point φ, λ coordinate, in degrees to four decimal fraction digits: Thus the ground resolution of coordinates is approximately 10 meters along the meridian. The file is compressed for distribution to 11.5 megabytes. (An animated planetography depiction of the data set can be seen [here].

Programs in this section use the same set of points, in a "flat" array of binary (8 bytes per point) UniSpherical coordinates. This specifies each point with an isometric ground resolution of between 5 and 15 millimeters, in a file of about 15.2 megabytes. (w1904711.p8b [Downoad]). The array is sorted in the numeric order of the point coordinate. (Note the singular: each point's location on the planetary surface is represented by a single unsigned integer number).

A short introductory program (listP8b.c - [source], [Linux executable] traverses the binary array and prints coordinates on the console. (For usage and functionalyty, see the commentary in program source).
